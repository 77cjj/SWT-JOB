# 适合人群

一句话摘要：判断你是否符合 SWT 参与条件，以及哪种参与模式更适合你。

## 硬性条件

| 条件 | 要求 |
|---|---|
| 年龄 | 年满 18 岁 |
| 学籍 | 在校大学生（含研究生） |
| 英语 | 能进行基础日常交流（敢开口比语法更重要） |
| 心态 | 能独立处理生活琐事与突发问题 |

## 两种常见参与模式

### 搞钱党（Hard Mode）

- 目标：尽量独立、控制成本、存下旅行资金。
- 特点：可能接受体力型岗位、考虑第二份工作、对住宿和开支更敏感。
- 适合：希望带回 3,000–6,000 美元净结余的同学。

### 体验党（Easy Mode）

- 目标：优先风景、文化体验与口语练习。
- 特点：对岗位舒适度要求更高，对净收入预期相对宽松。
- 适合：预算较充足、以交流体验为主的同学。

## 你需要具备的心理准备

- 中介只负责把你送出去，落地后的住宿、排班、纠纷需自行处理。
- 大部分时间在重复性基础岗位工作，不是纯旅游。
- 前期投入（中介费 + 机票）通常 2–3 万人民币，需有合理预期。
