# 在美就医与保险使用指南

适用于在美期间出现就医需求的同学，帮助你正确使用保险并控制医疗成本。

## 就医前先确认

| 项目 | 要点 |
|---|---|
| 保险有效期 | 确认就诊日期在保障范围内 |
| 医疗网络 | 优先选择保险网络内的医院/诊所 |
| 必备材料 | 护照、DS-2019、保险卡、联系方式 |
| 费用规则 | 了解自付额、报销比例、理赔时限 |

## 就医流程

1. 非紧急情况优先预约门诊（Primary Care / Urgent Care）。
2. 到院时出示证件与保险信息，确认是否网络内就诊。
3. 就诊后保留账单、诊断记录、处方与付款凭证。
4. 按保险要求在时限内提交理赔材料。

## 紧急情况处理

- 有生命危险：立即拨打 `911`。
- 非危及生命但需快速处理：前往 `Urgent Care` 更常见且成本通常低于 `ER`。
- 使用急诊（ER）前要理解：费用高、保险审核严格，不符合条件可能拒赔。

## 买药指南

### 非处方药（OTC）

- 常见购买渠道：Walmart、Target、Costco、CVS、Walgreens。
- 轻症优先咨询药师，按说明书使用，不叠加同类成分药物。

### 处方药（Prescription）

- 需美国执业医生处方后在药房配药。
- 取药时核对药名、剂量、频次和禁忌。

## 常见风险与建议

- 不要把“有保险”理解为“完全免费”，先看自付规则。
- 账单和理赔材料必须留存，避免后续争议。
- 涉及慢性病或长期用药，提前准备英文病史和药名清单。
