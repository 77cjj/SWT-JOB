# 美国公交出行指南

适用于在美国没有私家车、需要日常通勤或短途出行的同学。

## 适用场景

- 住宿与工作地点距离可通过公交覆盖。
- 想控制通勤成本，不频繁打车。
- 已确认所在城市有稳定公交线路。

## 出发前准备

| 项目 | 建议 |
|---|---|
| 线路查询 | 使用 Google Maps 或当地公交官网查询线路与时刻 |
| 票价准备 | 预留零钱或提前确认是否支持刷卡/App 支付 |
| 通勤时段 | 先确认首班车、末班车和高峰间隔 |
| 备用方案 | 夜间或错过班次时准备打车备选 |

## 乘车流程

1. 在地图中输入起终点，确认车次和换乘站点。
2. 提前到站等车，核对线路编号和方向。
3. 上车后按当地规则完成购票/刷卡。
4. 到站前按车内提示请求下车（如拉绳或按铃）。
5. 下车后及时确认返程线路与时刻。

## 乘车注意事项

- 小城市班次可能较少（例如 40-60 分钟一班），错过需等待较久。
- 个别站点信息简化，建议全程看地图定位。
- 不建议逃票，违规成本通常高于票价。
- 夜间通勤尽量选择照明良好、人员较多的站点。

## 携带自行车时

- 许多公交支持前置车架，一次可放 1-2 辆自行车（以当地规定为准）。
- 上下车时注意车辆与行人，避免阻塞通道。
