# Skype 使用指南

适用于收到 Skype 面试或沟通邀请的同学，重点覆盖安装、入会、音视频调试与国际通话注意事项。

## 适用场景

- 使用场景：与雇主视频面试、紧急语音沟通。
- 设备建议：优先电脑端，手机端用于备用。
- 账号准备：建议使用邮箱注册，便于找回密码。

## 下载与安装

| 端 | 下载方式 | 说明 |
|---|---|---|
| 电脑端 | `https://www.skype.com/zh-Hans/` | 建议从官网下载安装 |
| 手机端 | 应用商店搜索 `Skype` | iOS 可能受地区商店限制 |

## 面试前准备

1. 完成账号注册并确认 `Skype ID`。
2. 提前添加面试官账号，避免临时搜索出错。
3. 提前 10 分钟测试麦克风、摄像头和网络稳定性。
4. 使用安静环境，背景尽量简洁。

## 会议中操作

- 接听视频邀请：收到来电后选择视频接听。
- 音频控制：不发言时静音，减少杂音。
- 画面控制：摄像头保持开启并正对面部。
- 文本沟通：必要时用聊天框补充关键信息。

## 国际通话（可选）

- Skype 账号间语音/视频通常免费。
- 拨打普通电话需充值并关注费率规则。
- 拨号格式：国家区号 + 号码（美国 `+1`，中国 `+86`）。

## 注意事项与风险

- 不要通过不可信渠道共享账号密码。
- 如需第三方代充，避免泄露隐私信息。
- 与面试官联系时，先准备沟通提纲，确保表达清晰。
