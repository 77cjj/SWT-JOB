## 赴美通讯准备

同学们到美国一下飞机就要使用电话和网络，比如给家人报平安、联系雇主、打车、查地图等等，这就需要我们有能在美国进行通讯的工具。那么大家赴美前一定会有以下几个问题：

去美国该带什么手机？

国际漫游还是买美国电话卡？

选择哪个运营商？

买电话卡需要注意什么？

## 今天小西就给大家讲讲赴美前的通讯准备

去美国该带什么手机？

目前同学们的手机都是比较先进的智能机，基本上带到美国都可用。

但是也有个别品牌的手机在美国可能用不了，赴美前一定要查询好。

办国际漫游还是买美国电话卡？

1. 国际漫游：

目前国内运营商的美国漫游套餐资费都非常的贵，在美国信号不太好，有的套餐还需要交上千的押金，只适合几天左右的短期旅行用。所以我们不推荐办理国际漫游。

2. 美国电话卡：

我们在这里推荐大家办理美国电话卡，可以在网上提前购买。一般是一张SIM卡，卡里有一些预存的话费和通话套餐。卡激活后即可使用，话费用完了可以充值。

选择美国哪个运营商？

相比中国现有移动、联通和电信三大运营商。美国的运营商就多很多了，既有全国性的大运营商，也有地方性的小运营商，还有不少虚拟运营商。

1. 地方性的运营商：
有很多，如US cellular、LEAP等。基本就别考虑了，除非你对你要去的地方无比熟悉。

2. 全国性运营商：

全美有四大运营商，分别是AT&T、Verizon、Sprint和T-Mobile。

Verizon：其网络覆盖多，网速也快。但是基本上Verizon都卖的是无卡手机，且话费较贵。所以我们不推荐。

Sprint：信号覆盖差一些，在大城市没问题，小地方就差很多了，上网速率也比较慢。所以我们不推荐。

AT&T：网络覆盖率高，网速快，但价格稍高推荐！

T-Mobile：网络覆盖率较高，价格便宜，但是上网速度没有AT&T快。推荐！

3. 虚拟运营商：

所谓虚拟运营商，就是本身自己不建设网络，它从大运营商那儿批发一些号码，然后零售给用户，打上自己的品牌。这在国际上是很普遍的，美国常见的虚拟运营商有H2O、Lyca、Red Pocket、Ultra、Spot、Simple等。

虚拟运营商的网络覆盖和上网速率相对较差，但价格非常便宜。如果只在大城市里转悠，不担心电话打不通，不用随时随地上网，不使用网络地图导航，只想有个通信有备无患，在人多露天场合上上网，可以考虑虚拟运营商的电话卡，其他场合，还是算了吧……

（网上购买的美国电话卡，如果没有注明AT&T、T-Mobile等原生卡或写着其他公司名字，价格便宜的卡基本都是虚拟运营商的卡）。

买美国手机卡该注意什么？

1. 信号覆盖：

一定要问清自己工作地点是否有通话、网络信号覆盖（是否有2g/3g/4g）。

2. 激活方式：

有的美国电话卡一般需要提前几天激活，可提前知道电话号码，到达美国之后方便国内亲友与自己联系。

也有部分特殊套餐在国内激活后无法获知号码，到了美国才会以短信形式接收到本机号码。

3. 套餐内容：

根据自己的行程和使用量，决定通话时长、数据流量、开卡有效期。比如7天卡、15天卡、30天卡等。

同学们在美国会呆3个月以上，所以套餐可以组合着买，这里给大家一些建议：

1. 在国内网上购买好三个月以上套餐的电话卡，但是资费比较贵；

2. 也可以每个月换一张卡，这样比较便宜，缺点是换卡后号码会换；

3. 先购买一张7天或30天的套餐卡，在抵美后方便联系家人或雇主。等安顿好住宿后同学们大多数时间都是在使用wifi，不太用电话卡，这样就可以更换更便宜的套餐，或者是在美国当地直接购买更优惠的电话卡。

4. 上网设置：

有的手机插卡后可能需要设置接入点（APN）才能连接上网，建议在收到手机卡之后先对使用说明、卡正的反面进行拍照，到达目的地后，根据使用说明配置网络。

好啦，知道上面的内容，你就可以开始准备了。小西最后还有一个重重重点要提醒大家：请务必能让国内西米的辅导老师和家人随时能联系上你。
