# 生活成本 (Cost of Living)

控制成本是提高净收入的关键。美国的生活成本差异巨大，取决于你所在的州和生活方式。

## 必要支出预算 (Monthly)

| 项目 | 预估费用 | 说明 |
| :--- | :--- | :--- |
| **住宿 (Housing)** | $400 - $800 | 雇主提供通常较便宜，自己租房较贵。部分雇主直接从工资扣除。 |
| **食物 (Food)** | $200 - $400 | 自己做饭 ($200) vs 天天外食 ($600+)。 |
| **交通 (Transport)** | $0 - $100 | 最好住得离工作近 (步行/骑车)。Uber/Lyft 非常贵。 |
| **话费 (Phone)** | $30 - $50 | 推荐 Visible, Mint Mobile 等廉价套餐，或组建 Family Plan。 |
| **保险 (Insurance)** | $0 - $50 | 基金会通常包含基础保险，但看病可能有 Deductible (免赔额)。 |

## 避坑指南

### 1. 住宿坑
- **押金**：入住前检查并拍照记录房间状况，避免退房时被扣押金。
- **床虫 (Bed Bugs)**：美国廉价住宿常见问题，入住第一件事翻开床垫检查黑点。

### 2. 交通坑
- **没有车寸步难行**：如果工作地点偏僻且不提供班车，打车费会吃掉你一半工资。选岗时务必确认交通方式。
- **自行车**：Walmart 买辆便宜自行车 ($100左右) 是最划算的通勤方式，离开时可以卖给下一届或二手店。

### 3. 食物坑
- **外卖**：美国外卖不仅运费贵，还要给司机小费，尽量避免。
- **小费文化**：堂食必须给 15%-20% 小费，打包 (To-go) 可以不给或少给。
