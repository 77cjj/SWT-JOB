# 项目收支概览

一句话摘要：赴美前需准备多少费用、在美期间主要开支，以及合理收入预期。

> 数字来自 Sponsor 公开说明与常见机构报价，不是对任何个人的承诺；请以你签约的 Sponsor 与岗位为准。

## 适用场景

- 人群：计划参加 SWT 的在校大学生
- 阶段：报名决策期、行前预算期
- 目标：建立清晰的收支预期，避免资金缺口

## 赴美前费用（国内）

| 类别 | 明细 | 参考金额 |
|---|---|---|
| 国内机构费 | 咨询、材料审核、面试辅导 | 约 ¥3,000 |
| 项目费 | Sponsor 签发 DS-2019、保险、监管等 | 约 $2,300 |
| 护照 | 首次办理 | 约 ¥120 |
| 签证费 | J-1 非移民签证 | $185（约 ¥1,350） |
| 国际机票 | 因季节和航线差异较大 | ¥3000–¥6000 |
| **合计** | | **约 ¥23,000–¥28,000** |

> 具体以各机构当年报价为准，SEVIS 费（$35）通常含在项目费中。

## 赴美后费用（美国）

| 类别 | 说明 | 参考 |
|---|---|---|
| 初期现金 | 雇主通常 2 周后发薪，需自备 | $1,000–$1,500 |
| 住宿 | 因岗位和城市差异大 | $300–$800/月 |
| 餐饮 | 自己做饭可大幅节省 | $200–$400/月 |
| 交通 | 公交/拼车/自行车 | $50–$200/月 |
| 电话卡 | 见 [电话卡与通讯](/docs/departure/sim-card) | $30–$50/月 |

## 收入预期

| 场景 | 时薪参考 | 月工时 | 月 gross 估算 |
|---|---|---|---|
| 基础岗位 | $10–$15 | 160–200h | $1,600–$3,000 |
| 含小费岗位 | $2.13 + tips | 160–200h | 波动较大，旺季可观 |
| 第二份工作 | 视岗位 | 额外 80–120h | 额外 $800–$1,500 |

**合理净结余目标：** 认真工作 + 控支 + 可选二工，带回 **$3,000–$6,000** 并非不可能。

## 注意事项

- 以上均为参考区间，实际因州、岗位、个人消费习惯差异很大。
- 出发前根据 Job Offer 中的时薪和预估工时做个人预算。
- 详见 [生活成本](/docs/living/living-cost) 和 [机构与费用](/docs/apply/agency)。
