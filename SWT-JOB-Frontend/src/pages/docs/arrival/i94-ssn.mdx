# I-94 与 SSN 办理

一句话摘要：I-94 是入境记录，SSN 是报税与正式就业的关键号码——两者宜在激活 SEVIS 后尽快办理。

> 先完成 [SEVIS 激活](/docs/arrival/sevis-checkin)，再办 SSN。

---

## 下载 I-94

官方网站：[https://i94.cbp.dhs.gov/I94/#/home](https://i94.cbp.dhs.gov/I94/#/home)

1. 进入网站，选择「Get Most Recent I-94」。
2. 填写姓名、生日、护照号、签发国家（与护照一致）。
3. 下载 PDF 并**打印一份**，与护照、DS-2019 一起保管。
4. 后续办 SSN、驾照、部分银行开户都会用到。

### 常见问题

- **查不到记录**：入境后等待 24–72 小时再试；仍无记录联系 Sponsor。
- **姓名拼写错误**：按护照姓名在 SS-5 等表格中保持一致，必要时申请更正。

## 办理 SSN

### 材料清单

| 材料 | 是否必需 | 备注 |
|---|---|---|
| 护照与 J-1 签证 | 必需 | 原件，复印件通常不受理 |
| I-94 记录 | 必需 | 建议打印 |
| DS-2019 原件 | 必需 | 与护照信息一致 |
| SS-5 表格 | 必需 | [ssa.gov/forms/ss-5.pdf](https://www.ssa.gov/forms/ss-5.pdf) |
| Sponsor 说明信 | 视要求 | 按 Sponsor 指引准备 |

### 办理流程

1. 确认 **SEVIS 已激活**。
2. 查询当地 [SSA 办公点](https://secure.ssa.gov/ICON/main.jsp) 是否接受 walk-in 或需预约（各办公室政策不同）。
3. 现场提交材料并填写 SS-5，**索取受理回执**。
4. 等待邮寄卡片，通常 **2–6 周**；收到前可用回执 + 护照办理部分手续。

## 注意事项

- 不要通过非官方渠道「代办 SSN」——均为诈骗。
- 收到 SSN 前勿随意透露个人号码；收到后同样避免发给陌生人。
- 姓名与护照不一致时，先更正 DS-2019 或护照，再申请 SSN。
