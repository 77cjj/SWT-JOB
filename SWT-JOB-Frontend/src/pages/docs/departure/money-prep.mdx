# 资金与银行卡

一句话摘要：赴美前携带多少现金、如何换汇，以及国际卡与美国本地银行卡办理要点。

## 携带现金

雇主通常 **2 周后** 发首笔工资，需自备初期开支：

| 用途 | 说明 |
|---|---|
| 临时住宿与打车 | 落地当天 |
| 预付部分住宿费 | 视雇主/房东要求 |
| 前两周生活用品 | 食品、日用品 |
| 工作服等 | 视岗位要求 |

**建议携带：$1,000–$1,500**（根据 Job Offer 具体规划）。

## 换汇

| 方式 | 说明 |
|---|---|
| 中国银行 | 推荐，需身份证，大额提前预约 |
| 机场兑换 | 可换零钞，但汇率低、排队长 |

## 国内国际银行卡

出发前办理带 **Visa / MasterCard** 标志的借记卡或信用卡。

**主要用途：**

- 网上预订酒店、机票、大巴
- 网上购物与支付
- 现金存入卡中更安全（强烈推荐）
- 接收父母转账

**注意事项：**

- 尽量使用本人名下卡，名字不符可能被要求出示证明
- 预留父母或国内亲友电话（收验证码、挂失）
- 各银行各支行政策不同，办理前电话咨询

## 美国本地银行卡

抵美后需办本地银行账户收工资。

### 办卡材料

| 材料 | 必需 |
|---|---|
| 护照 | 是 |
| DS-2019 | 是 |
| Job Offer | 辅助 |
| SSN | 视银行要求 |

### 账户类型

| 类型 | 说明 |
|---|---|
| Checking Account | 活期，取钱无手续费，无利息 |
| Savings Account | 类似定期，存钱有利息，转出可能收费 |

### 关键区别

- 美国 **Account Number（账号）** 用于收汇款，**Card Number（卡号）** 用于刷卡
- 补卡后卡号会变，账号不变
- 刷卡通常只需卡背面 3 位 CVV，**妥善保管卡面信息**
- 离境前 **注销账户**，避免产生不良信用

### 支票 (Check)

雇主发工资可能以支票形式发放：

- 到 ATM 或银行柜台兑换
- 或用银行 App 拍照存支票（反面需签名背书），约 1 日到账
