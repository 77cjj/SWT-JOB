# 常见岗位详解

SWT 项目中提供的岗位通常是入门级的基础工作 (Entry Level)，不需要太高的专业技能，但对态度和体力有要求。

## 餐饮类 (Food & Beverage)

### Server (服务员)
- **工作内容**：点餐、上菜、收银、处理客诉。
- **收入**：底薪极低 + **高额小费**。旺季一天小费 100-300 美元不等。
- **要求**：英语口语流利，反应快，抗压能力强。

### Busser (传菜/收桌)
- **工作内容**：清理桌子、倒水、协助 Server。
- **收入**：底薪 + 分享部分小费 (Tip out)。
- **特点**：作为 Server 的跳板，压力相对小一些。

### Kitchen Staff / Dishwasher (后厨/洗碗)
- **工作内容**：备菜、烹饪、洗碗、清洁厨房。
- **收入**：固定较高时薪，无小费。
- **特点**：不用面对客人，适合社恐，但环境较差。

## 住宿类 (Hospitality)

### Housekeeping (客房清洁)
- **工作内容**：铺床、吸尘、清洁浴室、补充客房用品。
- **收入**：固定时薪 + 少量房间小费。
- **特点**：**体力活**。通常早上8点开工，下午3-4点结束，晚上有大量时间找第二份工。

### Front Desk (前台)
- **工作内容**：办理入住/退房、接听电话、解答咨询。
- **收入**：固定时薪。
- **特点**：英语提升最快，工作体面，不用晒太阳/干苦力。

## 游乐与休闲 (Amusement & Recreation)

### Ride Operator (设施操作员)
- **工作内容**：操作过山车等设施、检查安全带、维持秩序。
- **收入**：固定时薪。
- **特点**：在游乐园工作氛围欢快，但长时间站立晒太阳。

### Lifeguard (救生员)
- **工作内容**：泳池/水上乐园监护、施救。
- **收入**：固定时薪（通常略高于平均）。
- **特点**：需要通过红十字会考核（游泳+急救）。责任重大。

## 零售与销售 (Retail & Sales)

### Sales Assistant (销售助理)
- **工作内容**：理货、收银、导购（冰淇淋店、纪念品店等）。
- **收入**：固定时薪。
- **特点**：轻松，特别是在冷气房里的工作。
