# 签证材料清单

一句话摘要：J-1 面签所需全部材料一览，按自备/机构提供分类。

## 适用场景

- 阶段：收到 DS-2019 后、面签前
- 目标：确保材料完整，避免因遗漏导致拒签或补材料

## 核心材料

| 材料 | 必需 | 来源 | 备注 |
|---|---|---|---|
| 护照原件 | 是 | 自备 | 有效期距抵美 6 个月以上 |
| 签证照片 | 是 | 自备 | 51×51mm 白底，6 个月内拍摄 |
| DS-160 确认页 | 是 | 自备 | 在线填写，激光打印，条码清晰 |
| DS-2019 表 | 是 | 机构 | 签字在横线上方，避免折损 |
| SEVIS 缴费收据 | 是 | 机构 | 通常含在项目费中 |
| Job Offer 原件 | 是 | 自备 | 打印 |
| 基金会助签信 | 是 | 机构 | 打印 |
| 签证申请费收据 | 是 | 自备 | 中信银行或网银缴费凭证 |
| 签证预约确认信 | 是 | 机构 | 激光打印，条码清晰 |

## 辅助材料（自备）

| 材料 | 必需 | 备注 |
|---|---|---|
| 户口本原件 | 是 | 户口迁校者取户籍页/证明 |
| 身份证原件及复印件 | 是 | A4 纸正反面复印 |
| 学生证原件 | 是 | 丢失请尽快补办 |
| 在校证明及放假证明 | 是 | 学校抬头纸，中英文，盖章 |
| 无犯罪记录证明 | 是 | 保卫科或派出所，中英文 |
| 学校彩页/官网首页 | 是 | 彩色打印 |
| 成绩单 | 是 | 上一学期，教务处盖章 |
| 资金证明 | 是 | 父母银行流水（近 6 个月）+ 可选固定资产 |
| 父母在职证明 | 是 | 单位抬头纸，真实信息 |
| 全家福/生活照 | 是 | 近期，3 人左右，勿修图 |

## 禁止带入使馆

笔记本电脑、手机、手表、相机、U 盘、小刀、打火机、罐装饮料、背包/不透明公文包。

## 注意事项

- 每人准备一个文件夹，每项资料单独文件袋。
- 资金证明展示正常收入即可，不要炫富。
- 面签前联系项目负责老师核对清单完整性。
