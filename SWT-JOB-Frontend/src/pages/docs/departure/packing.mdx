# 行李清单 (Packing List)

出发前的最后准备，确保你带齐了重要物品，同时也轻装上阵。美国物资丰富，很多日用品去 Dollar Tree 或 Walmart 购买更划算。

## 必备材料清单

| 材料 | 是否必需 | 用途 | 获取方式 | 备注 |
|---|---|---|---|---|
| 护照 | 必需 | 出入境身份核验 | 个人证件 | 检查有效期与签证页 |
| DS-2019 原件 | 必需 | 入境与项目身份文件 | Sponsor 签发 | 建议随身携带原件 |
| Job Offer | 必需 | 雇主与岗位信息证明 | 机构/雇主提供 | 建议打印版 + 电子版 |
| SEVIS 费用收据 (I-901) | 必需 | 签证/入境辅助文件 | 在线缴费后下载 | 建议打印版 |
| 保险单 | 建议 | 医疗与理赔凭据 | 保险机构提供 | 电子版与纸质版都保留 |
| 往返行程单 | 建议 | 到达与返程信息核验 | 航司/平台下载 | 海关或接机沟通可用 |
| 驾照与翻译件 | 条件必需 | 租车/驾车 | 个人准备 | 仅在有驾车需求时携带 |

## 随身小包 (Carry-on Personal Item)
重要的证件和贵重物品务必贴身携带，**千万不要托运**。

- [ ] **护照** (Passport) - 确认签证页有效
- [ ] **DS-2019 表格原件** (非常重要！没有这个无法入境/办理社安号)
- [ ] **Offer (Job Offer Form)** - 打印件
- [ ] **SEVIS 费用收据** (I-901 Receipt) - 打印件
- [ ] **保险单** (打印件或电子版存手机)
- [ ] **现金 + 银行卡** (建议携带 $500-$1000 现金，大额钞票建议去超市破开；Visa/Mastercard 信用卡)
- [ ] **驾照** (中国驾照 + 翻译件，如果打算租车)
- [ ] **笔** (飞机上填写入境卡用)
- [ ] **手机 SIM 卡取卡针**

## 背包 35L (Carry-on Backpack)
*建议尺寸：高62cm x 宽23cm x 深22cm (需符合航司登机标准)*

### 电子产品
- [ ] **电脑 + 充电器** (如果打算在美国买新的，可以带个旧的或者不带)
- [ ] **便携副屏** (如果有工作/学习需求)
- [ ] **手机** (建议备用机) + **充电头** + **数据线** (多带几根，美国数据线较贵)
- [ ] **充电宝** (磁吸/大容量，注意额定能量不超过 100Wh)
- [ ] **转换插头** (美标，两扁一圆)

### 旅途用品
- [ ] **水杯** (过安检后接水)
- [ ] **U型枕 / 眼罩 / 耳塞** (长途飞行三件套)
- [ ] **薄外套/连帽衫** (飞机上真的很冷)
- [ ] **充饥食物** (巧克力/能量棒)
- [ ] **洗漱小样** (一次性牙刷、小样洗面奶，用于转机/飞机上)

## 托运行李 (Checked Luggage)
*建议准备 28寸 或 26寸 行李箱。注意检查航司额度（通常是 2件 x 23kg）。*

### 生活用品
*大部分可以在美国买，带急用的即可。*
- [ ] **牙刷牙杯** (建议带旅行套装，美国牙刷刷头通常很大)
- [ ] **护肤品** (神经酰胺/凡士林 - 美国气候干燥，保湿很重要；防晒霜)
- [ ] **毛巾** x3 (速干款更好)
- [ ] **指甲刀 / 剪刀**
- [ ] **小锁** (宿舍柜子可能需要)
- [ ] **便携餐具** (筷子/勺子，美国主要用刀叉)
- [ ] **眼镜/隐形眼镜** (美国配镜很贵，建议多备一副)
- [ ] **针线包** (小型的即可)

### 衣物类
*SWT 工作通常需要站立很久，鞋子舒服最重要。*
- [ ] **换洗衣服** (短裤/T恤 3套，够换洗即可，美国买衣服便宜)
- [ ] **内衣裤/袜子** (6套+)
- [ ] **工作裤** (很多雇主要求 Black Slacks 黑色西裤/休闲裤，建议国内买好带过去)
- [ ] **舒适的运动鞋** (一定要舒服！除了拖鞋外建议带两双)
- [ ] **拖鞋** (一双洗澡用)
- [ ] **舒适鞋垫** (站立工作神器，推荐足弓支撑款)
- [ ] **泳裤/泳镜** (海滩/泳池必备)
- [ ] **外套** (North Face/冲锋衣，早晚温差大，空调房冷)
- [ ] **轻薄手套** (橡胶手套，如果是做 Housekeeping 或后厨工作很有用)
- [ ] **口罩** (备用)

### 住宿与杂物
- [ ] **床上用品** (冰丝床垫/枕套 - 宿舍床通常是 Twin Size，建议提前问好尺寸，带轻便的)
- [ ] **美标插排** (宿舍插座可能很少，带一个线长的)
- [ ] **洗衣袋** (去公用洗衣房用)

### 药品 (Medicine)
*带说明书，不要带含麻黄碱的药（如白加黑/泰诺等部分感冒药需谨慎）。*
- [ ] **感冒药** (莲花清瘟等中成药/感冒灵)
- [ ] **退烧药** (布洛芬)
- [ ] **肠胃药** (蒙脱石散/整肠生 - 水土不服必备)
- [ ] **消炎药** (阿莫西林/头孢 - 美国处方药很难买，建议适量携带)
- [ ] **创口贴** (多带点)
- [ ] **抗过敏药**
- [ ] **跌打损伤膏药** (体力劳动可能会用到)

---

### ❌ 建议不带/少带
*   **锅碗瓢盆**：重且易碎，Walmart 很便宜。
*   **大量沐浴露/洗发水**：太重，带旅行装哪怕一两周的量即可。
*   **老干妈/调料**：亚超都有，除非极度偏僻。
*   **昂贵的首饰**：宿舍人多手杂，避免丢失。
