# 美国打车指南（Uber / Lyft / Taxi）

适用于在美国短途出行、夜间通勤或公交不便场景。

## 适用场景

- 临时出行、晚间返程、携带行李。
- 公交班次不足或换乘成本高。
- 需要快速到达机场、车站、雇主地点。

## 打车方式对比

| 方式 | 优点 | 注意点 |
|---|---|---|
| Uber / Lyft | 下单便捷、价格透明、可追踪行程 | 高峰期价格波动明显 |
| 传统 Taxi | 可电话预约，部分城市可路边拦车 | 小城市可用车辆较少 |

## App 打车流程

1. 下载并注册 `Uber` 或 `Lyft`。
2. 绑定支付方式（银行卡、电子支付等）。
3. 输入上车点和目的地，比较车型与预估价格。
4. 确认车辆信息后下单等待接驾。
5. 上车前核对车牌与司机姓名，行程结束后核对账单。

## 电话叫车（Taxi）流程

1. 搜索当地 `taxi company` 并拨打电话。
2. 提供姓名、手机号、上车地址和预约时间。
3. 询问是否支持信用卡支付和预计到达时间。
4. 记录车牌和司机联系方式后在指定地点等车。

## 安全与费用建议

- 上车前核对车牌、车型和司机头像，信息不一致不要上车。
- 尽量在光线好、人流较多的位置等车。
- 高峰时段先比价再下单，必要时切换平台。
- 不建议通过非官方代叫车渠道绑定个人支付账号。

## 常见问题

### 司机未找到你

- 在 App 内更新上车点并主动发送定位备注。
- 通过内置聊天联系司机，避免跨平台泄露隐私号码。

### 账单异常

- 保留行程截图和订单记录。
- 在 App 内发起费用申诉或退款请求。
