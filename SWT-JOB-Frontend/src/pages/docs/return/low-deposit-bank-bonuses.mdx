# 低存款门槛银行开户奖励（SWT 向）

> 奖励金额与条件随时变化，**以银行官网与条款为准**。下文为整理思路，不构成金融建议；F-1/SWT 身份、有无 SSN、是否 US Person 会影响可开账户类型。身份概念可参考 [NRA / RA / US Person 区分](https://www.uscreditcardguide.com/resident-alien-non-resident-alien-us-person-green-card/)（外部站点）。

## 为什么 SWT 要关心「低 DD 门槛」

很多 checking 开户奖励要求 **Direct Deposit（工资直存）** 达到一定金额。SWT 学生：

- 第一份工作前可能没有稳定 DD；
- 有些雇主用 paper check 或 prepaid 卡发薪；
- 存款不多时，更适合先看 **低门槛、可 online 入金** 的 checking/savings 促销。

更全的奖励汇总类信息可在社区维护的 [Checking / Savings / MMA Bonus Summary](https://www.uscreditcardguide.com/checking-savings-mma-bonus-summary/) 对照（外部链接，请自行核实时效）。

## 选型原则（比背银行名单更重要）

1. **KYC 身份**：无 SSN 时能否开户、是否需 ITIN/稍后补 SSN——以银行实时政策为准。
2. **DD 定义**：是否必须 employer payroll code；Zelle/ACH 自行转账有时不算。
3. **Funding**：能否用 debit card / ACH 从已有账户小额入金激活。
4. **Clawback**：奖励到账后若过早关户可能被扣回。
5. **税务**：开户奖励通常 taxable，留 1099-INT 等表格。
6. **ChexSystems**：短期开太多 checking 可能影响后续开户。

## 常见「相对低门槛」方向（需自行核实当前 offer）

| 类型 | 特点 | SWT 注意点 |
|------|------|------------|
| 线上 neobank / fintech（如 Chime 类） | 常配合 refer；DD 门槛相对灵活 | 见站内 [薅羊毛](/deals) 与 [副业与羊毛](/docs/return/side-hustles) |
| 大型银行 checking 促销 | 有时 $500–$1000 DD 档 | 可能需要 SSN、地址证明 |
| Credit union 本地 promo | 有时低门槛 | 可能需要本地地址/现场 |
| Savings / MMA 阶梯利率 | 不一定是「开户红包」 | 适合已有一定存款者 |

**不要**为了奖励同时开过多账户；优先选 **1–2 个** 能覆盖发薪与日常消费的账户。
