# 结束后旅行 (Grace Period Travel)

J-1 签证结束后，你有 **30 天的合法停留期 (Grace Period)**。这一个月是你在美国最自由的时光，可以畅游美东美西。

## 热门路线规划

### 1. 美西公路旅行 (West Coast Roadtrip)
*   **路线**: 洛杉矶 (LA) -> 拉斯维加斯 (Vegas) -> 大峡谷 (Grand Canyon) -> 旧金山 (SF) -> 1号公路。
*   **适合**: 租车自驾。
*   **亮点**: 1号公路海景、Vegas 赌场、大峡谷壮丽景色。

### 2. 美东城市人文 (East Coast)
*   **路线**: 纽约 (NYC) -> 费城 (Philadelphia) -> 华盛顿特区 (DC) -> 波士顿 (Boston)。
*   **适合**: 公共交通 (Amtrak/Megabus)。
*   **亮点**: 博物馆、百老汇、历史名胜。

### 3. 南部风情
*   **路线**: 奥兰多 (Orlando) -> 迈阿密 (Miami) -> Key West。
*   **适合**: 喜欢海滩和游乐园 (Disney/Universal)。

## 旅行省钱技巧
1.  **住宿**: 使用 **Hostelworld** 订青旅，或多人拼 **Airbnb**。
2.  **交通**: 提前买 **Megabus** 或 **Flixbus** 车票，只要几刀。
3.  **门票**: 使用 **CityPASS** 通票，比单买便宜 40%。

## 注意事项
*   **Grace Period 只可以出境，不可以再次入境**。如果你去了墨西哥或加拿大，就无法再回美国了（除非有B1/B2签证）。
*   **离境时间**: 必须在 DS-2019 结束日期后的 30 天内离境，切勿滞留。
