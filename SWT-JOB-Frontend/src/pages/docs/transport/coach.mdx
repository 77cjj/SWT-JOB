# 美国长途巴士出行指南

适用于跨城市出行、预算有限且时间相对灵活的同学。

## 常见平台

| 平台 | 官网 | 覆盖特点 |
|---|---|---|
| Greyhound | `www.greyhound.com` | 覆盖广，经典长途线路多 |
| Megabus | `us.megabus.com` | 东北部与中西部较常见 |
| RedCoach 等区域平台 | 各平台官网 | 区域内点对点线路 |

## 购票流程

1. 在官网输入出发地、目的地和日期。
2. 比较不同班次的总时长、中转次数和票价。
3. 选择票种并确认行李规则、改签退票政策。
4. 填写乘客信息并完成支付。
5. 保存电子票与订单号，出发前再次核对站点信息。

## 票务与行李重点

| 项目 | 需确认内容 |
|---|---|
| 票种 | 是否可改签、是否可退票、改退手续费 |
| 行李 | 免费件数、超重收费、托运行李尺寸 |
| 取票方式 | 电子票、打印票或站内取票 |
| 乘车站点 | 站点名称是否相似，避免走错站 |

## 乘车安全建议

- 上车后保管好证件和贵重物品，不离身。
- 中途休息下车时注意发车时间，避免误车。
- 夜间或中转等待时尽量在车站明亮区域停留。
- 不随意帮陌生人看管行李或代带物品。
