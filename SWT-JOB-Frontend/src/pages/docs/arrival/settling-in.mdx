# 首周安顿

一句话摘要：完成身份手续后，核对保险、整理证件、熟悉通勤，进入稳定工作节奏。

## 雇主报到

- 按 Job Offer 日期到岗，携带护照、DS-2019、I-94 复印件（部分雇主需留存）。
- 确认排班表、制服要求、 breaks 规则。
- 索要 HR 或主管直接联系方式（WhatsApp / 电话）。

## 保险核对

- 抵美后 **48 小时内** 获取保险卡（实体或电子）。
- 核对：生效日期、免赔额、网络内医院、是否覆盖 Grace Period 旅行。
- 就医前确认诊所是否 **in-network**，避免高额账单。

## 证件清单（建议统一文件夹）

| 证件 | 保管建议 |
|---|---|
| 护照 + J-1 签证页 | 原件随身或锁柜，复印件分存 |
| DS-2019 | 原件勿污损，出入境必带 |
| I-94 | 打印件 + 电子备份 |
| 保险卡 | 钱包放一张，复印件备一份 |
| SSN 卡 / 受理回执 | 收到 SSN 前用回执；收到后 SSN 勿随意携带 |
| 雇主 Offer / 合同 | 电子版 + 纸质备份 |

## 风险提示

- SSN、护照等敏感信息不要随意外传（含网友、不明「代办」）。
- 证件姓名与护照不一致时，立即申请更正。
- 更换住址或岗位时，**第一时间**更新 Sponsor Check-in 与雇主记录。
