# 美国骑行安全指南

适用于通勤距离中短、希望降低交通成本的同学。

## 适用场景

- 住处到工作地点 3-8 公里范围内。
- 城市有可骑行道路或慢行通道。
- 愿意遵守当地交通规则并配备基础安全装备。

## 购车或租车建议

| 方式 | 特点 | 注意点 |
|---|---|---|
| 租车 | 初期成本低，短期灵活 | 需确认押金、维修责任、还车规则 |
| 购车 | 长期更划算 | 需要额外购买锁具和照明配件 |

## 骑行前检查清单

- 头盔：尺寸合适并固定稳固。
- 车灯与反光：夜间前白后红，保证可见性。
- 刹车与轮胎：出发前快速检查。
- 路线：提前规划，尽量避开高速和复杂路段。

## 骑行规则与安全要点

1. 遵守交通信号，不闯红灯、不逆行。
2. 转向时提前打手势并观察后方来车。
3. 与停靠车辆保持安全距离，避免“开门杀”。
4. 夜间尽量减少骑行，必须骑行时穿高可视服饰。
5. 骑行中不看手机、不戴降噪耳机，保持环境感知。

## 风险提示

- 各州对头盔、车灯和车道规则可能不同，出行前先查当地规定。
- 与机动车混行时，优先保证可见性和预判空间。
- 发生事故时第一时间报警并保留现场照片与联系人信息。
