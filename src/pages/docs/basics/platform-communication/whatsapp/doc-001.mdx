# WhatsApp 使用指南

适用于需要和雇主、房东或海外联系人保持即时沟通的同学，覆盖安装、注册和常见问题处理。

## 适用场景

- 海外联系人主要使用 WhatsApp 沟通。
- 需要文字、语音、视频和文件传输。
- 需要与电脑端同步处理消息。

## 下载与安装

| 端 | 官方下载方式 | 说明 |
|---|---|---|
| 手机端 | [https://www.whatsapp.com/download](https://www.whatsapp.com/download) | 建议优先使用官方渠道 |
| 电脑端 | [https://www.whatsapp.com/download](https://www.whatsapp.com/download) | 需配合手机端登录 |

## 注册流程

1. 打开 WhatsApp，选择国家并输入手机号。
2. 接收短信验证码并完成验证。
3. 设置昵称与头像，完成账号初始化。
4. 按需开启联系人权限，用于快速添加联系人。

## 电脑端登录

- 电脑端 WhatsApp 需要通过手机端扫码绑定。
- 登录后消息可在手机和电脑端同步。

## 常见问题

### 收不到验证码

可能原因：
- 网络状态不稳定。
- 手机拦截了海外短信/电话。
- 账号触发了平台风控。

处理建议：
1. 关闭短信和来电拦截后重试。
2. 选择语音电话验证码。
3. 更换网络后再次请求验证码。

## 注意事项

- 请勿通过来路不明的网站下载安装包。
- 账号涉及隐私沟通，建议开启两步验证。
- 与陌生人沟通时避免发送证件原图和敏感信息。
