# 赴美饮食与做饭指南

适用于刚到美国、希望兼顾饮食健康与生活成本的同学。

## 适用场景

- 日常以工作通勤为主，做饭时间有限。
- 需要在“外食方便”和“自己做饭省钱”间平衡。
- 想快速找到可靠餐厅和超市渠道。

## 日常就餐策略

| 方案 | 适合人群 | 成本特征 |
|---|---|---|
| 快餐/连锁店 | 时间紧张 | 便捷但长期成本偏高 |
| 本地餐厅 | 想体验当地文化 | 质量差异大，需看评价 |
| 自己做饭 | 控制预算与口味 | 前期采购稍多，长期最省 |

## 找餐厅的实用工具

- `Google Maps`：看营业时间、距离和评分。
- `Yelp`：看真实评价和菜品图片。
- `OpenTable`：热门餐厅预约。
- 外卖平台：按配送范围和费用筛选。

## 餐厅点餐与结账

1. 先点饮品，再点主菜（必要时加前菜）。
2. 不确定菜品可直接问服务员配料和分量。
3. 结账前确认是否分单（split bill）。
4. 堂食通常有小费规则（常见 15%-20% 区间）。

## 自己做饭采购清单

| 类别 | 建议采购 |
|---|---|
| 主食 | 米、面、面包、速冻主食 |
| 蛋白 | 鸡蛋、鸡肉、牛肉、豆制品 |
| 蔬果 | 叶菜、根茎类、耐储水果 |
| 调味 | 盐、糖、酱油、油、基础香料 |
| 应急 | 速食、罐头、常温奶、饮用水 |

## 做饭安全注意事项

- 做饭时保持通风，避免油烟触发烟感报警。
- 电炉加热慢，提前预热，防止空烧锅具。
- 生熟分开处理，避免交叉污染。
- 冰箱食品按日期管理，过期及时丢弃。

## 省钱建议

- 先做一周菜单再采购，减少冲动消费。
- 关注超市会员价和周折扣。
- 可与室友拼单大包装食材，分摊成本。
