# 常见误区

一句话摘要：澄清关于 SWT 项目性质、收益与中介职责的常见误解。

## 误区对照

| 误解 | 实际情况 |
|---|---|
| 「这是一个赚钱项目」 | 能赚钱，但前期投入高；若纯为赚钱，国内实习可能性价比更高 |
| 「这是一个旅游项目」 | 工作占大部分时间，Grace Period 才是纯旅行窗口 |
| 「中介会安排好一切」 | 中介负责申请与签证辅导，落地后的生活需自行解决 |
| 「参加完 J-1 两年内不能去美国」 | 谣言，详见 [常见问题 · 212e](/docs/intro/faq#212e-条款与再次赴美) |
| 「口语必须像雅思 7 分才能去」 | 日常沟通能力更重要，面试重在表达真实意图 |

## 关于收益的预期

- **底线**：覆盖中介费与机票（回本）。
- **合理目标**：认真工作 + 合理控支，带回 3,000–6,000 美元并非不可能。
- **变量**：岗位时薪、工时、是否二工、所在州生活成本。

## 关于工作性质

SWT 岗位多为入门级基础工作（餐饮、零售、酒店前台、救生员等），对态度和体力要求高于专业技能。选岗决定收入潜力、英语锻炼机会和社交圈子，需提前研究。
