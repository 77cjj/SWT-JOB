# 税务与退税总览

美国税务对 J-1 SWT 学生来说分两条线：**工资单代扣（Payroll Withholding）** 和 **次年报税（Tax Return）**。前者决定每两周到手多少，后者决定多退少补。

> 本文档提供**全美通用硬性约束**；各州差异见 [薪资代扣详解](/docs/return/payroll-withholding) 与分州附录。

## 全美硬性约束（Federal）

### 1. FICA（Social Security + Medicare）

| 项目 | 税率 | J-1 NRA 硬性约束 |
| --- | --- | --- |
| Social Security | 6.2%（有年薪封顶） | **NRA 身份应为 0%**（IRC §3121(b)(19)） |
| Medicare | 1.45%（无封顶） | **NRA 身份应为 0%** |

- 持 **J-1 / F-1** 且税务身份为 **Nonresident Alien (NRA)** 的前 5 年，**不应**被扣 FICA。
- 若 HR 系统将身份误设为 Resident，Payroll 会自动扣 **7.65%**——应立刻联系 HR 更正。
- 多扣的 FICA **不能**通过常规 1040-NR 退税，需 Form 843 或雇主修正（过程漫长，**尽量在发薪时解决**）。

### 2. 联邦所得税（Federal Income Tax）

- 依据 **W-4** 与年化收入推算，采用累进税率（通常 10%–12% 区间对 SWT 学生常见）。
- **NRA 硬性约束**：不得勾选 Standard Deduction 相关优惠。
- 若提交 **Form 8233** 并获 HR 批准，可按 **Tax Treaty** 在年化收入中先扣减免税额（如中美协定 **$5,000**），再计算预扣。
- 次年通过 **Form 1040-NR** 多退少补。

### 3. 州所得税（State Income Tax）

- **全美 9 个无州所得税州**：AK, FL, NV, NH（仅利息股息）, SD, TN, TX, WA, WY。
- 其余州各有独立税率表与表格（如 NJ-1040NR、CA 540NR）。
- **关键边界**：多数州**不承认**联邦 Tax Treaty 免税额——州税预扣通常以 Gross 为基数。

## 净收入估算公式

```
Net Pay = Gross
        − FICA（NRA 应为 0）
        − 州强制扣款（Disability / UI 等，视州而定）
        − Federal Income Tax
        − State Income Tax
        − 其他（401k、保险等，SWT 通常无）
```

**经验值**：在 FICA 正确豁免、非高税州的情况下，总扣税约 **10%–15%**（不含不可退的州强制扣款）。

## 退税时间与流程

1. **1 月底前**：雇主发放 **W-2**（或 1042-S，视身份与 treaty 而定）。
2. **1–4 月**：使用 **Sprintax**（推荐 NRA）或专业中介申报；TurboTax 默认不支持 NRA。
3. **4–8 周**：IRS / 州税局退款到账（联邦通常可退大部分预扣；州税视州而定）。

> **警告**：不报税可能影响未来签证。即使预计无退款也应申报。

## 常见问题

**Q：为什么预扣税比实际应税额高很多？**  
A：Payroll 按「全年 52 周 × 本期工资」年化估算。若你只工作 3 个月，预扣会偏高，**次年报税可退回**。

**Q：NJ Disability / UI 能退吗？**  
A：不能。这是州立法强制社保基金，属于 Statutory Fee，非所得税。

**Q：在哪里核对我的身份设置？**  
A：联系雇主 HR，确认系统中 Tax Status = **Nonresident Alien**，且 W-4 / 8233 已正确提交。
