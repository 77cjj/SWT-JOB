# 赴美工作规则与上岗口语

适用于在美工作期间处理排班、请假、二工、换岗，以及日常与主管、同事沟通。

> 相关文档：[面试与口语准备](/docs/apply/interview) · [第二份工作](/docs/living/second-job) · [邮件沟通模板](/docs/departure/email-templates)

## 工作期间基本责任

- 按 `DS-2019` 规定时间合法工作，不擅自超期或提前离岗。
- 遵守雇主排班、着装和行为规范。
- 生病或无法到岗时，尽量提前通知主管并同步 Sponsor。
- 任何工作或住址变化，先报 Sponsor 再执行。

## 上岗操作步骤

### 首日报到（Day 1）

1. 按 Offer 或邮件要求的时间到 HR / 前台报到，携带护照、DS-2019、I-94 等（以雇主清单为准）。
2. 领取工牌、制服，确认更衣室和休息区位置。
3. 见直属主管（Supervisor），确认本周排班表。
4. 问清：打卡方式、吃饭休息规则、紧急联系人电话。
5. 当天结束前记下 Sponsor 值班邮箱/电话，保存到手机。

### 日常上班流程

1. 提前 **10–15 分钟** 到岗，换好制服。
2. 打卡（Clock in）后找主管确认当日任务。
3. 午休或休息结束前回到岗位；下班前打卡（Clock out）。
4. 收到排班变更、加班或调岗通知，当场确认时间并保留记录（短信/邮件）。

### 请假 / 生病不能上班

1. 尽早通知主管（越早越好，至少提前数小时）。
2. 说明原因 + 预计能否次日返岗，例如：`I'm not feeling well and can't come in today.`
3. 同步联系 Sponsor（按项目要求）。
4. 保留请假记录；连续多天病假可能需要医生证明（以雇主政策为准）。

### 与同事 / 主管发生冲突

1. 先冷静，不在客人面前争执。
2. 私下沟通：`I'd like to talk when you have a moment.`
3. 仍无法解决 → 联系主管的上级或 HR → 再联系 Sponsor。
4. 重要对话尽量留邮件或文字记录。

## 第二份工作（二工）流程

1. 先确认不影响第一份工作排班。
2. 获取意向雇主信息（岗位、地址、工时、联系人）。
3. 向 Sponsor 提交申请与所需材料。
4. 收到 Sponsor **书面批准** 后再正式入职。

## 换工作与时间变动

- 主动换工作：需先说明理由并等待 Sponsor 审核。
- 被动换岗/换雇主：保留沟通记录，配合 Sponsor 协调。
- 到岗或离岗时间变动：需雇主和 Sponsor 双方同意。

## 分场景上岗口语

### 报到与排班

| 情境 | 推荐口语 |
| :--- | :--- |
| 首日报到 | Hi, I'm [Name]. I'm the new J-1 student. I'm here for my first day. |
| 确认排班 | Could you show me my schedule for this week? |
| 不懂任务 | Could you show me how to do this? I want to make sure I do it correctly. |
| 请求重复 | Sorry, could you repeat that more slowly? |

### 餐饮 / 前台 / 零售

| 情境 | 推荐口语 |
| :--- | :--- |
| 对客人 | Hi, how are you today? / I'll be right with you. |
| 点单没听清 | Sorry, could you say that again? |
| 结账 | Your total is [amount]. Cash or card? |
| 忙不过来 | I'm on my way. / Give me one moment, please. |

### Housekeeping / 清洁

| 情境 | 推荐口语 |
| :--- | :--- |
| 敲门进房 | Housekeeping. / May I come in? |
| 房间有人 | Sorry to disturb you. I'll come back later. |
| 缺用品 | We're out of [towels/linens]. Could I get more from the storage room? |
| 完成打扫 | I'm finished with room [number]. |

### 请假与调班

| 情境 | 推荐口语 |
| :--- | :--- |
| 当天请假 | Hi [Supervisor], I'm not feeling well and can't make it today. I'm sorry for the short notice. |
| 提前请假 | I'd like to request [date] off. Is that possible? |
| 想换班 | Could I swap my shift on [day] with someone? |
| 确认收到 | Got it, thank you. I'll be there at [time]. |

### 向 Sponsor 汇报问题

| 情境 | 推荐口语（邮件/电话均可） |
| :--- | :--- |
| 工作问题 | I have an issue at my workplace and need guidance. My employer is [name]. |
| 住址变更 | I need to report a housing change. My new address is [address]. |
| 二工申请 | I'd like to apply for a second job. Here are the employer details: [info]. |

## 推荐练习

上岗前或首周，每天练 **15 分钟** 即可明显改善沟通信心。

### 练习 1：打卡全流程（5 分钟）

模拟从进门到下班，大声说一遍：

`Good morning` → `Clocking in` → `What's my task today?` → `I'm going on break` → `Clocking out, see you tomorrow`

### 练习 2：岗位情景卡（10 分钟）

按你的岗位从上面表格选 **5 个情境**，每个说 2 遍：

- 第 1 遍看稿
- 第 2 遍只看中文情境，英文脱口而出

### 练习 3：请假电话模拟（1 次/周）

用手机录音，模拟给主管打电话请假 30 秒，回听是否：

- 说明了原因
- 语气礼貌
- 没有过长解释

### 练习 4：与 AI / 同学角色扮演

让同伴或 AI 扮演 Supervisor，随机问：

- `Can you work this Saturday?`
- `Why were you late?`
- `A guest is complaining. What do you do?`

你用英文回答，练即兴反应。

## 常见场景处理

| 场景 | 处理原则 |
| :--- | :--- |
| 与同事/雇主沟通冲突 | 先内部沟通，无法解决再联系 Sponsor |
| 找第二份工作 | 不影响主工作，且必须先获 Sponsor 批准 |
| 岗位调整 | 与雇主和 Sponsor 同步确认后执行 |
| 提前结束项目 | 及时说明原因，按 Sponsor 指引办理 |

## 风险提示

- 未经批准私自二工或换工作，可能触发项目终止风险。
- 被雇主辞退后，应第一时间联系 Sponsor 获取正式处置指引。
- 重要沟通尽量保留邮件证据，避免只有口头信息。
- 对客人可使用友好口语，但对主管和同事仍需保持职业礼貌。
