# 美国住宿指南

适用于需要自行寻找或确认住宿的同学，涵盖找房、签约、入住和退租关键点。

## 适用场景

- Job Offer 未提供住宿，或仅提供可选住宿信息。
- 需要短租（通常 3-4 个月）并控制预算。
- 需要通勤可达、合同可控的住房方案。

## 选房流程

1. 明确入住与退租日期、预算和通勤时长上限。
2. 通过雇主、Sponsor 和公开平台收集房源。
3. 与房东确认房型、租期、费用和押金规则。
4. 审核并签署 `Housing Lease`（租约）。
5. 入住前拍照留档，退租时按合同结算押金。

## 房型对比

| 房型 | 适合人群 | 典型特点 |
|---|---|---|
| Apartment / Suite | 2-4 人合租 | 空间完整，成本较高 |
| Single Room | 注重隐私 | 单独卧室，共享公共区 |
| Shared Room | 预算敏感 | 卧室多人，成本较低 |
| Dormitory | 接受集体居住 | 床位式，管理规则较多 |

## 租房前必查清单

| 项目 | 需确认内容 |
|---|---|
| 费用结构 | 租金是周付还是月付，是否含水电网 |
| 押金规则 | 押金金额、可退条件、扣费标准 |
| 合同约束 | 最短租期、提前退租违约条款 |
| 通勤条件 | 到工作地距离、夜间交通可达性 |
| 入住状态 | 家具家电清单、损坏责任界定 |

## 找房渠道

- 雇主/Offer 信息：优先确认历史学生常用住房点。
- Sponsor 支持：可咨询当地合作住宿资源。
- 社区平台：`Facebook Groups`（按城市 + J1/SWT 关键词）。
- 房源网站：`Craigslist`、`Airbnb` 等（注意信息甄别）。

## 订房与签约注意事项

- 任何付款前先确认房东身份与房源真实性。
- 尽量保留邮件或平台聊天记录，避免口头约定无凭据。
- 入住当天对房屋现状拍照/录像并发给房东确认。
- 退租前按合同要求完成清洁和钥匙交接，减少押金争议。

## 短期酒店补充（过渡住宿）

- 入住前确认入住/退房时间和取消政策。
- 关注额外费用：清洁费、停车费、服务费、押金预授权。
- 注意禁烟规定和设备损坏责任。
