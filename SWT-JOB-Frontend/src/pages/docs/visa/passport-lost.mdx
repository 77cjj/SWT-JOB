# 护照遗失处理

一句话摘要：在美国遗失护照后的报案、补办与旅行证申请流程。

## 适用场景

- 在美工作或旅行期间护照遗失、被盗或损毁
- 时间允许：向就近中国使领馆 **补发护照**（约 30 天）
- 急于回国：申请 **旅行证**（约 2 个工作日，仅限回国使用）

## 处理步骤

### 1. 警察局报案

1. 携带可用身份证件（驾照、学生证等）到当地警察局报案。
2. 记录护照号码、国籍、遗失时间地点等信息。
3. 获取正式 **Incident Report**（非仅报案小卡片），正常约 1 周、$10；紧急情况可请求加急。

> 复印件在此环节极为重要——Incident Report + 护照复印件 + 其他身份证明可用于搭乘航班前往领馆。

### 2. 向使领馆申请

**旅行证申请材料：**

| 材料 | 是否必需 |
|---|---|
| 护照/旅行证申请表 | 必需 |
| 近期证件照 3 张（48×33mm，白底） | 必需 |
| 原护照复印件或国籍身份证明 | 必需 |
| 情况说明（简况、遗失经过、联系方式等） | 必需 |
| 遗失声明（确认找回后不可再用） | 必需 |
| Incident Report | 必需 |

**费用参考：**

| 类型 | 费用 |
|---|---|
| 普通件 | $35/件 |
| 加急 | +$20 |
| 特急 | +$30 |

付款方式：Master Card / Visa 或 Money Order（不接受现金、个人支票）。

### 3. 领取证件

- 按取证单日期或电话通知，本人或委托他人凭取证单原件领取。
- 办妥一年未取证，证件将被作废。
- 原护照即使找回也不可继续使用，应送交使领馆注销。

## 紧急联系

- 中国领事保护热线：**+86-10-12308**（按 0 再按 9 转人工）
- 各领馆 24 小时领保电话见 [外交部官网](http://www.fmprc.gov.cn)

## 预防措施

- 出发前复印护照、签证、DS-2019 各 2 份，分开放置。
- 手机存证件照**不能替代**复印件——政府机关只认纸质 copy。
