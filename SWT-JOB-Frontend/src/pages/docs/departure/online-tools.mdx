# 线上面试工具

一句话摘要：Google Meet、Skype、Zoom、Microsoft Teams 四款线上面试工具的入会方式与面试前检查清单。

## 适用场景

- 阶段：收到线上面试邀请后、面试前
- 设备建议：优先电脑端 + Chrome 浏览器，手机作为备用
- 网络：提前确认稳定，准备热点备用

## 工具对比

| 工具 | 入会方式 | 推荐端 | 备注 |
|---|---|---|---|
| Google Meet | 链接 / 会议代码 / 日历邀请 | 电脑 Chrome | 最常见 |
| Zoom | 链接 / 会议 ID + 密码 | 电脑或 App | 需提前安装 |
| Skype | 联系人邀请 / 链接 | 电脑或 App | 注意国际通话设置 |
| Microsoft Teams | 链接 / 日历邀请 | 电脑或 App | 企业面试常见 |

## 通用入会前检查

无论使用哪款工具，面试前 10 分钟完成：

- [ ] 麦克风可正常收音
- [ ] 摄像头画面清晰，背景整洁
- [ ] 显示名称设为真实姓名或「英文名 + 姓氏」
- [ ] 关闭其他占网程序和其他会议软件
- [ ] 关闭消息提醒，避免弹窗干扰
- [ ] 准备纸笔记录关键信息

## Google Meet

- 访问 [meet.google.com](https://meet.google.com) 或点击邀请链接
- 会议代码入会：输入代码后点击加入
- 会议内：控制麦克风/摄像头开关，仅在要求时屏幕共享

## Zoom

- 安装 [zoom.us](https://zoom.us) 客户端或使用浏览器入会
- 链接入会：点击邀请链接，输入显示名称
- ID 入会：打开 Zoom → Join → 输入 Meeting ID 和密码
- 首次使用建议提前测试音视频

## Skype

- 安装 Skype 并注册/登录 Microsoft 账号
- 通过联系人邀请或会议链接入会
- 调试：Settings → Audio & Video 测试设备
- 国际面试注意时区差异

## Microsoft Teams

- 通过邀请链接或 Teams App 入会
- 首次使用可能需下载桌面客户端
- 会议内可使用聊天面板发送简短补充信息
- 仅在面试官要求时共享屏幕

## 注意事项

- 不要同时开启多个会议软件，避免音视频设备被占用
- 如遇卡顿，切换有线网络或手机热点
- 面试中保持摄像头开启（除非特殊要求）
- 详见 [面试准备](/docs/apply/interview)
