# 面签流程

一句话摘要：从 DS-160 填写到面签当天全流程，含使领馆地址与注意事项。

## 流程概览

| 步骤 | 内容 | 说明 |
|---|---|---|
| 1 | 填写 DS-160 | [ceac.state.gov/genniv](https://ceac.state.gov/genniv/) |
| 2 | 注册签证账户 | [ustraveldocs.com/cn_zh](http://www.ustraveldocs.com/cn_zh/index.html) |
| 3 | 缴纳签证费 | $185，中信银行网银或柜台 |
| 4 | 预约面签 | 提供 DS-2019 编号、DS-160 号、UID、CGI 号等 |
| 5 | 面签当天 | 携带材料、录指纹、面试 |
| 6 | 领取护照 | 约 1 周后邮寄至预留中信银行 |

## 面签当天流程

1. **寄存物品**：除签证材料外一律寄存（使馆附近有寄存处）。
2. **门卫检查**：核对护照与预约信息。
3. **材料检查**：秘书检查 7 样核心材料（护照、DS-2019、DS-160 确认页、签证费收据、SEVIS 收据、照片、Job Offer）。
4. **录指纹**：十指。
5. **面签**：在指定窗口等待，**统一用英文回答**。
6. **领取结果**：当场出结果；通过后检查 DS-2019 上签证官是否签字。

## 面签结果

| 结果 | 说明 |
|---|---|
| 通过 | 护照留馆办签，给通过信 |
| 拒签 | 返还护照和材料，给拒签信；分析原因后可再约 |
| 补材料 | 护照留馆，按单子要求补交 |
| 行政审核 | 护照留馆，等待进一步审查 |

## 面签注意事项

**着装：** 整洁得体，淡妆即可，不佩戴过多首饰。

**行为：**

- 不插队、保持安静、不与陌生人闲谈
- 面签官无论中英文，统一英文回答
- 自信、微笑、目光交流

**获签后：**

- 检查 VISA 页个人信息是否正确
- **等机构下发抵美行程后再购买机票**
- 复印扫描留存签证页
