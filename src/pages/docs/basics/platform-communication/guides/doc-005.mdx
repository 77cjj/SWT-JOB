# Zoom 使用指南

适用于使用 Zoom 参加线上面试的同学，覆盖安装、入会方式、面试前检查和常见问题处理。

## 适用场景

- 使用场景：雇主线上面试、远程沟通。
- 设备建议：优先电脑端，手机端为备用。
- 网络要求：需提前验证网络可稳定连接 Zoom 服务。

## 安装方式

| 端 | 下载方式 | 备注 |
|---|---|---|
| 电脑端 | `https://zoom.us/zh-cn/download` | 推荐用于面试主设备 |
| 手机端 | 应用商店搜索 `Zoom` | 作为临时备用方案 |

## 入会方式

1. **会议链接**：点击邮件中的 Zoom 链接直接入会。
2. **会议 ID 入会**：打开 Zoom，输入会议 ID（必要时输入密码）。

## 入会前检查

- 提前 10 分钟进入会议测试。
- 检查麦克风、摄像头、扬声器是否正常。
- 参会名称建议使用真实姓名或“英文名 + 姓氏”。
- 关闭与面试无关的应用和消息提醒。

## 会议中建议

- 不发言时保持静音。
- 除非面试官要求，避免频繁切换背景和设备。
- 共享屏幕时仅共享必要窗口，防止隐私泄露。

## 常见问题

### 无法入会或出现错误代码

- 优先检查网络是否稳定。
- 退出后重新进入会议链接。
- 切换设备（电脑与手机）交叉验证。

### 听不到声音或对方听不到

- 在 Zoom 音频设置中重新选择输入/输出设备。
- 检查系统级麦克风权限是否已授予。
