# 在美旅行规划指南

适用于工作结束后或休假期间的美国境内旅行规划。

## 出行前必做事项

| 项目 | 说明 |
|---|---|
| 证件 | 护照、签证、DS-2019、I-94 电子记录 |
| 保险 | 核对旅行期间是否覆盖，不覆盖需续保/另购 |
| 预算 | 交通、住宿、门票、餐饮、应急金 |
| 行程 | 明确城市顺序、停留天数和返程节点 |

## 同行与安全

- 建议 2-4 人结伴，避免单人夜间长距离移动。
- 不与陌生网友临时拼团或合住。
- 共享行程给可信联系人，保留紧急联系方式。

## 城市旅行模板（可复用）

1. 先确定“必去 2-3 个点”。
2. 以地理位置分区安排每天路线。
3. 预订门票和热门项目时段。
4. 留出机动时间应对天气和交通延误。

## 交通与住宿策略

| 场景 | 建议 |
|---|---|
| 跨州移动 | 优先飞机/长途巴士，提前锁低价票 |
| 城市内游玩 | 地铁/公交 + 步行，必要时打车 |
| 住宿选择 | 先看交通安全，再看价格和评价 |

## 费用控制

- 大件门票提前在线购买，避免现场高价。
- 酒店选择“可免费取消”方案，便于调价重订。
- 多人出行可用分账工具，减少结算摩擦。

## 风险提示

- 不把全部现金和证件放同一处。
- 夜间尽量避开治安复杂区域。
- 行李和贵重物品离身可控，避免寄放无保障地点。
