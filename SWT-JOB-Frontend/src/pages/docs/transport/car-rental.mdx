# 美国租车指南

适用于在美国短期自驾出行的同学，覆盖预订、取车、保险和还车流程。

## 租车前准备

| 材料/条件 | 是否必需 | 说明 |
|---|---|---|
| 驾照（本国或美国） | 必需 | 以当地州规定为准 |
| 驾照英文翻译/公证件 | 常见需要 | 便于门店核验 |
| 主驾驶名下信用卡 | 必需 | 用于押金与扣费 |
| 最低年龄要求 | 平台规则 | 低龄驾驶常有附加费 |

## 预订渠道选择

- 官网直订：条款清晰、售后直接、适合新手。
- 第三方比价：价格可能更低，但需核对隐藏费用。
- 重点核对：里程限制、异地还车费、机场附加费、附加驾驶员费。

## 取车流程

1. 到店出示证件并确认订单信息。
2. 核对合同中的保险和增值服务是否与预期一致。
3. 取车前拍摄全车外观和油量表，留存时间戳。
4. 确认是否“满油取还”、是否无限里程。

## 保险理解（简版）

| 保险类型 | 作用 |
|---|---|
| CDW/LDW | 保障租赁车辆受损风险 |
| Liability（责任险） | 保障对第三方人车财产责任 |
| PAI（人身意外） | 覆盖车上人员部分医疗风险 |
| 车内财产保障 | 覆盖部分随车财物损失 |

不同平台命名可能不同，以下单合同为准。

## 还车流程

1. 按约定地点和时间还车（超时可能按整天计费）。
2. 加满油后还车（若合同要求满油归还）。
3. 现场与工作人员确认车况与结算单。
4. 保存还车单据与扣费凭证，便于后续争议处理。

## 驾驶安全与合规

- 严格遵守限速、停车和校车规则。
- 不酒驾、不疲劳驾驶、不手持手机驾驶。
- 儿童乘车需符合儿童座椅法律要求。
- 突发事故优先保证人身安全，必要时报警 `911`。
