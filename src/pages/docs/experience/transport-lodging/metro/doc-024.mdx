# 美国地铁出行指南

适用于在有地铁系统的城市进行日常通勤或城市内出行。

## 覆盖城市与票务

| 城市示例 | 官方网站 |
|---|---|
| 纽约 | `http://www.mta.info/` |
| 华盛顿 | `https://wmata.com/` |
| 芝加哥 | `https://www.transitchicago.com/` |
| 波士顿 | `https://www.mbta.com/` |
| 旧金山 | `https://www.bart.gov/` |
| 洛杉矶 | `https://www.metro.net/` |

建议在对应官网确认运营时间、票价、是否支持交通卡和换乘规则。

## 乘坐流程

1. 在地图 App 输入起终点，确认线路方向和换乘站。
2. 进站前准备好交通卡或购票方式。
3. 看清站台方向（如 Uptown / Downtown）再上车。
4. 到站前提前靠近车门，下车后核对换乘信息。

## 纽约场景补充

- 纽约地铁存在快线（Express）与慢线（Local），同编号列车停靠站可能不同。
- 高峰期拥挤和延误较常见，建议预留 15-30 分钟缓冲。
- 站台无屏蔽门区域较多，候车时与边缘保持安全距离。

## 安全与效率建议

- 尽量避开早晚高峰出行。
- 夜间优先选择照明好、人流更多的站点和车厢。
- 不在站台边缘玩手机或佩戴降噪耳机。
- 信号弱路段提前保存路线和换乘截图。
