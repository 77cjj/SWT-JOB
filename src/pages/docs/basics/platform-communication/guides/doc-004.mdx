# Microsoft Teams 使用指南

适用于使用 Microsoft Teams 进行线上面试的同学，涵盖入会流程、面试前检查和会议内常用功能。

## 适用场景

- 使用场景：雇主线上面试、远程会议。
- 设备建议：优先电脑端（Chrome/Edge），手机端为备用。
- 入会方式：链接、会议 ID + 密码。

## 入会前准备

| 项目 | 建议 |
|---|---|
| 名称显示 | 使用真实姓名或“英文名 + 姓氏” |
| 音视频测试 | 提前 10 分钟测试麦克风与摄像头 |
| 网络环境 | 关闭占网应用，确保连接稳定 |
| 通知管理 | 手机和电脑关闭弹窗提醒 |

## 入会步骤

1. 点击邮件中的 Teams 会议链接。
2. 电脑端优先选择浏览器继续，或使用本地 Teams 客户端。
3. 进入预览页后检查名称、麦克风、摄像头。
4. 点击 `Join now` 进入等待区，等待面试官准入。

## 会议内核心操作

- 麦克风：不发言时静音，发言前再开启。
- 摄像头：除非明确要求关闭，建议保持开启。
- 屏幕共享：按需共享窗口或演示文稿，避免共享全桌面隐私信息。
- 聊天框：用于补充链接、文件或简短说明。
- 举手功能：多人场景下有序发言。

## 注意事项

- 使用背景虚化时，纸质材料可能无法清晰展示。
- 若浏览器不兼容，及时切换客户端，避免临场排障。
- 面试结束后确认已点击挂断，避免误占会话。
